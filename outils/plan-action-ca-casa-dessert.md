# Plan d'action CA — Casa Dessert

**Document préparé pour :** Casa Dessert (casadessert.fr)  
**Date :** mai 2026  
**Contexte :** livreur particuliers en place, comptable pour les factures pro, site + admin + onglet Livreur (Waze).

---

## 1. Où est votre CA aujourd'hui ?

Deux moteurs distincts :

| Canal | Client type | Levier principal |
|-------|-------------|------------------|
| **Particuliers** (site) | Soirée, livraison, panier moyen | Volume de commandes × panier moyen |
| **Pro** (restaurants, cantines…) | Récurrence, volumes | Nombre de clients actifs × fréquence × ticket |

Le livreur débloque surtout le **B2C** (fiabilité, zone élargie psychologiquement).  
Le comptable débloque le **B2B** (relances, encours, factures carrées — tableur Excel + alertes 500 €).

**Rappels opérationnels actuels :**
- Livraison particuliers : départements **75, 91, 92, 93, 94**
- Créneau affiché : **après 21h30**
- Hors Paris **75013** et à plus de **10 km** : minimum de commande **15 €**
- Promo site : **2 tiramisus achetés = boisson offerte**
- Outils : admin (commandes, stats), onglet **Livreur** (Waze), suivi factures pro, devis en ligne

---

## 2. Priorités sur 90 jours (ordre recommandé)

### Phase 1 — Stabiliser et mesurer (semaines 1–2)

Sans chiffres clairs, difficile d'augmenter le CA.

1. **Fixer 3 KPI hebdomadaires** (15 min le lundi) :
   - Nombre de commandes particuliers
   - CA particuliers (€)
   - CA pro (saisie admin + factures comptable)
2. **Panier moyen particulier** : objectif simple, ex. passer de X € à X+3 € (via box, 2e dessert, promo).
3. **Taux de clients pro « actifs »** : combien commandent au moins 1× / mois.

Votre admin (Stats + onglet Pro) sert pour le B2C ; le **Excel encours** + comptable pour le B2B.

---

### Phase 2 — Particuliers : convertir le livreur en machine à commandes (semaines 2–6)

Le livreur n'est rentable que si **plus de gens commandent** et **plus souvent**.

**Actions concrètes :**

1. **Communication locale ciblée** (budget faible, fort impact) :
   - Stories / posts : « Livraison ce soir après 21h30 » + zone (Paris + petite couronne).
   - Carte ou flyer QR → casadessert.fr dans les immeubles / commerces du 13e et alentours.
2. **Relance clients qui ont déjà commandé** (e-mails / téléphones en base) :
   - 1 message / semaine max : nouveau parfum, promo 2 tiramisus, « on livre chez vous ».
3. **Augmenter le panier moyen** :
   - Mettre en avant les **box** et un 2e produit (promo boisson offerte — la rappeler au checkout).
   - Seuil 15 € hors 13e : proposer un petit add-on si le panier est proche du minimum.
4. **Créneau soir = positionnement** : « dessert après le dîner / pour demain » — contenu Instagram 17h–19h.

**Objectif réaliste :** +20 à 30 % de commandes particuliers en 6 semaines si communication régulière chaque jour de production/livraison.

---

### Phase 3 — Pro : transformer les comptes en revenus récurrents (semaines 3–12)

C'est souvent là que le **plus gros CA** se joue (volumes, marge, moins de courses unitaires).

**Avec le comptable + votre suivi :**

1. **Règle des 500 €** : dès alerte encours → appel + mail le jour même (facture + RIB + date limite).
2. **Rythme commercial fixe** :
   - 1 visite / appel pro / mois pour les meilleurs comptes.
   - 1 relance devis (devis.php) pour les prospects sans commande.
3. **Offre pro simple** :
   - 3 parfums « rotateurs » du mois + prix pro clairs sur la boutique.
   - Prépaiement / acompte **500 €** pour sécuriser la trésorerie.
4. **Nouveaux pros** : salons de thé, traiteurs, hôtels boutique, événementiel IDF — **2 nouveaux contacts qualifiés / semaine**.

**Objectif réaliste :** +1 client pro actif par mois ou +15 % sur les comptes existants.

---

### Phase 4 — Offre et prix (en continu, 1 décision / mois)

1. **1 nouveau parfum ou édition limitée / mois** → raison de recommander.
2. **Revoir les prix pro** dans l'admin (marge après livraison + temps comptable).
3. **Événements** : commandes groupées (bureau, anniversaire) = paniers 80–150 €.

## 3. Plan semaine par semaine (résumé)

| Semaine | Particuliers | Pro | Admin / outils |
|---------|--------------|-----|----------------|
| 1 | KPI + 3 posts « livraison ce soir » | Liste encours avec comptable | Livreur : tournée Waze |
| 2 | Relance 10 anciens clients | Relancer comptes > 500 € | Stats admin : top saveurs |
| 3 | Flyer / QR quartier | 1 rdv pro | Ajuster stocks best-sellers |
| 4 | Promo box visible checkout | 1 nouveau devis → commande | Bilan CA mois 1 |

Puis répéter le cycle en augmentant l'intensité commerciale.

---

## 4. Ce qu'il ne faut pas faire (garder la marge)

- Élargir la zone de livraison trop vite (coût livreur + temps).
- Baisser les prix pro pour « faire volume » sans seuil de commande minimum.
- Trop de parfums en stock → focus sur ce qui tourne (Stats admin).

## 5. Objectifs CA indicatifs (à ajuster avec vos chiffres)

Exemple si aujourd'hui **~2 000 € / mois particuliers** et **~3 000 € / mois pro** :

| Horizon | Particuliers | Pro | Total mensuel |
|---------|--------------|-----|---------------|
| 3 mois | +25 % | +15 % | ~7 000 € |
| 6 mois | +40 % | +30 % | ~10 000 € |

À recalculer avec le CA réel des 3 derniers mois.

## 6. Prochaines étapes pour affiner le plan

Pour un plan **30-60-90 jours** chiffré, noter :

1. CA moyen / mois particuliers et pro (ordre de grandeur).
2. Nombre de commandes / semaine actuelles.
3. Nombre de clients pro actifs.
4. Capacité de production max (barquettes / soir).

## 7. Pistes techniques (site / outils)

Évolutions possibles sur casadessert.fr :

- E-mails automatiques après commande (confirmation + relance).
- Page ou message « commander pour demain ».
- Accès **livreur** dédié (sans mot de passe admin complet).
- Export CA mensuel pour le comptable.
- Rappels encours pro automatisés par e-mail.
